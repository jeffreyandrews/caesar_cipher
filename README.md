# caesar_cipher
This program is a simple caesar cipher tool that I created to develop a better understanding of encoding important information. This program requires python 3 and should be run using caeser_cipher.py in the terminal. This program prompts the user to enter a password and the number of shifts they would like. Using the shift number, the program encoded the user's password to keep it secure. The user can then enter their password where they get 5 attempts to do so, and if they get it right they can view their encoded password. Through this project I strengthened my knowledge of basic encryption systems and how to keep important information classified. It also helped me strengthen my understanding of how to use flag variables and loops to easily perform the desired task. I would like to incorporate my password strength checker into this program to have them work together to create a full working password system.
